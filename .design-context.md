# Design Context

## Brand Personality
- **Primary**: Premium Dark AI assistant.
- **Tone**: Sophisticated, fast (low latency), and editorial.
- **Style**: Modern, using Plus Jakarta Sans for authority and Inter for readability.

## Target Audience
- Tech-savvy users looking for private, local LLM interaction.

## UI Requirements
- **Core Architecture**: Sliver-based components (NestedScrollView, SliverAppBar, SliverList).
- **Aesthetics**: Glassmorphism, tonal hierarchy, vibrant gradients (Indigo to Cyan).
- **Responsive**: Mobile-first, desktop-optimized (constrained content width).

## Accessibility
- WCAG 2.1 AA target. Dark mode contrast-first.
