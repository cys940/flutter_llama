# 브랜드 로고 및 스플래시 화면 디자인 설계서

## 1. 개요
`assets/images/digital_curator_logo`를 앱의 공식 브랜드 로고로 채택하고, 모바일 환경에서 전문적인 첫인상을 주기 위해 네이티브 및 Flutter 레벨의 하이브리드 스플래시 화면을 구현합니다.

## 2. 디자인 요구사항
- **로고**: `assets/images/digital_curator_logo` (메인 브랜딩 요소)
- **배경색**: 기존 다크 테마(`#121212`)와 조화를 이루는 색상
- **구성 요소**: 로고(중앙) + 앱 이름(하단)
- **적용 대상**: Android, iOS (모바일 전용), 앱 내부 UI (모두)

## 3. 상세 설계 (하이브리드 방식)

### 3.1 네이티브 설정 (Native Branding)
- **패키지**: `flutter_launcher_icons`, `flutter_native_splash` 사용
- **설정 내용**:
    - 스플래시 배경색 지정
    - 중앙 로고 배치 (고해상도 대응)
    - 하단 앱 이름 텍스트 포함형 이미지 구성

### 3.2 Flutter UI 통합 (UI Integration)
- **로고 위젯 (`AppLogo`)**:
    - `lib/core/widgets/app_logo.dart`에 독립 위젯으로 구현
    - `flutter_animate`를 적용하여 등장 시 페이드인 및 약한 바운스 효과 부여
- **주요 배치**:
    - **Sidebar**: 채팅 내역 사이드바 최상단 제목 영역 대체
    - **Header**: 메인 채팅 화면 AppBar 좌측에 소형 아이콘 배치
    - **Settings**: 앱 정보(About) 섹션에 버전 정보와 함께 로고 노출

## 4. 검증 계획
- 네이티브 앱 실행 시 초기 스플래시 화면이 테마와 일치하는지 확인 (화이트 플래시 현상 제거)
- 앱 내부 로고 위젯이 화면 크기에 맞춰 적절히 렌더링되는지 확인
- 애니메이션 효과가 부드럽게 작동하는지 확인
